## Supported Distributions

The `bootstrap.sh` script supports multiple Linux distribution families with automatic detection:

- **Debian / Ubuntu** – uses `apt`
- **Fedora / RHEL / CentOS** – uses `dnf` (falls back to `yum`)
- **Arch / Manjaro** – uses `pacman`
- **openSUSE** – uses `zypper`
